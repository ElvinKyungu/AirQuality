# Air quality application

Une application qui vous donne l'air de votre ville, si l'air est pollué ou plutôt bon.

## Installation

Utilisez le gestionnaire de paquets npm pour installer l'application.

```bash
npm install
npm run dev
```

## Technologies

```
Vue js
CSS
```
